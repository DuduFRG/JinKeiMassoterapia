# Jin Kei Massoterapia Oriental — Site Corporativo

Site desenvolvido para apresentação aos donos da Jin Kei Massoterapia Oriental.

## Estrutura de arquivos

```
jinkei/
├── index.html          ← Página inicial (Home)
├── sobre.html          ← Sobre a clínica
├── servicos.html       ← Todos os 16 serviços
├── depoimentos.html    ← Avaliações e depoimentos
├── contato.html        ← Contato, mapa e formulário
├── sitemap.xml         ← Sitemap SEO
├── robots.txt          ← Diretivas para mecanismos de busca
├── README.md           ← Este arquivo
└── assets/
    ├── css/
    │   └── style.css   ← Toda a estilização
    ├── js/
    │   └── main.js     ← Animações e interações
    └── img/
        └── logo.png    ← Logo Jin Kei

```

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `jinkeimassoterapia`)
2. Faça upload de todos os arquivos desta pasta
3. Vá em Settings → Pages → Branch: `main` → pasta: `/ (root)`
4. O site estará disponível em `https://SEU-USUARIO.github.io/jinkeimassoterapia/`
